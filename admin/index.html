🤖 Claude API (Dialogue AI)

Required for the Dialogue AI tab. Get your key at console.anthropic.com. Stored locally — never sent anywhere except Anthropic.

💡 Alternatively, set ANTHROPIC_API_KEY in Netlify environment variables for server-side security.

🖼️ Google AI (Image AI)

Required for the Image AI tab. Get your key at aistudio.google.com. Stored locally only.

💡 Alternatively, set GOOGLE_AI_KEY in Netlify environment variables.

🎬 Replicate (Podcast Video / Lip-Sync)

Required for the Podcast Video tab. Sign up at replicate.com → Account Settings → API tokens. Free $5 credit on signup.

💡 Alternatively, set REPLICATE_API_KEY in Netlify environment variables.
